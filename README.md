# ATTS
O projekcie

To repozytorium GitHub jest częścią projektu ATTS – Axolotls Tech Team Support.
ATTS to grupa technologiczna zajmująca się tworzeniem i rozwijaniem:

pluginów do Minecrafta,

skryptów i narzędzi wspomagających,

modyfikacji (modów) i zasobów technicznych,

materiałów nagraniowych oraz dokumentacji związanej z Minecraftem.

Celem projektu jest wspieranie społeczności poprzez dostarczanie stabilnych, nowoczesnych i łatwych w użyciu rozwiązań technicznych.
